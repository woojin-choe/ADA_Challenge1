# 남는 시간
남는 시간은 포항에 처음 살게 된 러너들을 위해 기획하고 제작하게 된 앱입니다.
조금씩 짧게 짧게 남는 시간들을 의미있게 보내고 싶은데 포항에서 무엇을 할 수 있는지 몰라 시간을 낭비하는 러너 없애는게 목표입니다.

## 주요 기능
- 당장 남는 시간 선택
- 활동 카테고리 선택
  - 운동
  - 산책
  - 공부
  - 요가
  - 휴식
  - 등등 추가될 예정
- 추천 활동 목록 제공
- 위치 검색 및 주변 장소 기반 선택 화면 제공

## 화면 구성

### 1. Splash View
앱의 첫 화면

### 2. Home View
사용자가 현재 남은 시간을 선택할 수 있는 메인 화면이다.  
검색창과 원형 타이머 UI를 중심으로 구성되어 있으며, 아래에서는 추천받고 싶은 활동 카테고리를 선택할 수 있다.

### 3. Location Sheet
위치 찾기 화면이다.  
장소 또는 주소를 검색할 수 있으며, 지도 UI를 통해 현재 위치나 주변 지역을 바탕으로 활동을 선택할 수 있도록 구성하였다.

### 4. Recommend View
선택한 조건에 맞추어 활동을 추천해주는 화면이다.
앞서 선택한 주소 주변에서 할 수 있는 활동을 지도에 표현해서 나랑 어느정도 거리에 있는 활동인지 확인할 수 있도록 구성하였다.
활동 이미지, 제목, 간단한 설명을 카드 형태로 제공하여 사용자가 빠르게 비교하고 선택할 수 있다.

## 기술 스택
- Swift
- SwiftUI
- Xcode
- MapKit
- CoreLocation
- Property Wrapper

## 디자인 방향
전체적으로 차분하고 정제된 분위기의 UI를 목표로 하였다.  
밝은 배경과 블루 계열 포인트 컬러를 활용하여 안정감과 신뢰감을 주고,  
복잡한 조작 없이도 사용자가 바로 추천을 받을 수 있도록 직관적인 흐름으로 구성하였다.

## 향후 개선 방향
- 위치 기반 추천 정확도 향상
- 활동 추천 알고리즘 고도화
- 사용자 맞춤형 추천 기능 추가
- 즐겨찾기 및 최근 활동 기록 기능 추가
- 캘린더/일정 연동 기능 추가
